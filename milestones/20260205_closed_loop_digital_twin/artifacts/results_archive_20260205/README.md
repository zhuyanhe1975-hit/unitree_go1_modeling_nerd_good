# Archive: results/ exports

这些文件来自开发过程中的本地目录 `results/`，为方便追溯与复现，将其中的**图片结果**与**相应注解（summary markdown）**归档到本里程碑下。

- `png/`：评估/诊断图（reversal、full-cycle、tau_pd vs tau_Nm 等）
- `md/`：对应的评估汇总与过程注解（`summary_closed_loop_csv*.md` 等）

说明：`results/` 目录本身默认不进 git（见 `.gitignore`），因此需要长期保留的关键结果请放在本目录或其他 `milestones/<date>_<name>/` 下。
